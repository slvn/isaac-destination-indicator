# isaac-destination-indicator

Add to the HUD the destination of your current run.
  - Shift + 1 : Blue Baby
  - Shift + 2 : The Lamb
  - Shift + 3 : Mother
  - Shift + 4 : The beast (currently using mega satan asset)
  - Shift + 0 : (clear)

# More
